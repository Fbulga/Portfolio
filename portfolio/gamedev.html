Goose Game (2024)

Developer

Goose Game is a low-poly 3D stealth game where you play as a duck on a mission to steal a golden banana from a museum packed with guards and security cameras.

The main goal of this project was to learn how to develop NPCs using finite state machines and decision trees to create AI with complex behaviors. It involved implementing custom pathfinding using the A* algorithm to guide the NPCs through the environment efficiently.
